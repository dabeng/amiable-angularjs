Sum up what Angular brings to us

Angular considered every aspect of front-end for us

It means that you can build the whole front-end section of web application with only one js framework. There is no need to learn any other vendor stuff.

Change your inertial tinking of building layout

With the help of ngBind, ngShow, ngSwitch, especially ngClass, you can easily adjust layout breaking away from verbose, low level dom manipulation code.

Allow us to build more lively prototype system based on backend-less development

Using the ngResource module to build Restful APIs consumer on client-side and using ngMockE2E to build RESTful APIs providers, we'll get more rich and interesting front-end section.

Feel free to reuse your existing code

No matter whether a page portion or a dom fragment like a button group, you can easily encapsulate them into the directives for cross-application reuse.

Countless components and modules on top of AngularJS waiting for you to choose

If it's not your goal to invent a new js framework, please put more effort in accomplish business logic with few, clean and elegent code.

Allow us to test every corner in the front-end code

In fact, just following AngularJS basic usage, we do produce the testable front-end code.
