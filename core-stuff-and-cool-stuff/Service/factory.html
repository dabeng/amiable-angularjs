A factory is an injectable function.

Username retrived from authentication factory: {{ dataFromFactory }}

Instruction

  • A factory is basically a function that returns an object representing the recipe.
  • We could have some variables private to the factory and some other logic not directly visible from the service object instance.

Boilerplate of creating factory

            
//define a factory using factory() function
app.factory('myFactory', function () {
 
  var serviceObj = {};
  serviceObj.variable1 = someValue1; 
  serviceObj.variable2 = someValue2; 

  serviceObj.method1 = function () {
    //TO DO:
  };
 
  serviceObj.method2 = function () {
   //TO DO:
  };
 
 return serviceObj;
});
